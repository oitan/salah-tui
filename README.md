# salah-tui

Terminal UI for tracking the five daily prayers, reminders, and quick status logging.

# Status

MVP scaffolding. Everything is subject to change before `v1.0.0`.

# Install

## Prerequisites

- Go >= 1.25.4
- Git

## From source

```bash
git clone https://github.com/oitan/salah-tui
cd salah-tui
make build
```

# Usage

```bash
./bin/salah-tui # run TUI
```

## Keybindings (default)

* `q`: quit
